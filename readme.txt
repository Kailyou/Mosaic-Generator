1. The tool contains two different kinds of source-tiles
	- Example tiles (loaded by default)
	- Own tiles (Also example tiles included)
		- Settings --> Load own Tiles --> choose 70x75 pixel and press okay

2. tool has been tested on windows 7 (64 bit) and windows 10 (64 bit)

3. Clicking on "Generate mosaic" might cause some loading time depending on the size of the image.

4. Settings --> "Show Mosaic Image" needs to be pressed to show the mosaic image

5. Do not forget to uncheck the "Show Mosaic Image" while trying to load a new image.
